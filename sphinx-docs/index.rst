MDiocre
=======
MDiocre is a Python tool for creating static webpages and
websites using Markdown source files.

Have a look at my terrible `code <_modules/index.html>`_!

Requirements:
   * pandoc   (program needed, pypandoc loads this)
   * pypandoc (for conversion)
   * colorama (for terminal output)
